anchors

Description:

[ NoAccessorMethod ]
public abstract string anchors { construct set; owned get; }

The path to a file containing PEM encoded certificate authority root anchors.

The certificates in this file will be treated as root authorities for the purpose of verifying other certificates via the verify_chain operation.
