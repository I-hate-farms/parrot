get_state

Description:

public abstract Variant get_state ()

Queries the current state of this.

If the action is not stateful then null will be returned. If the action is stateful then the type of the return value is the type given by get_state_type.

The return value (if non-null) should be freed with g_variant_unref when it is no longer required.

Parameters:

this

a Action

Returns:

the current state of the action
